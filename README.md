# Sistema de Ranking de Indicações

Sistema web para visualizar o ranking dos top 10 clientes por indicações.

## Como usar
1. Acesse o site
2. O ranking será exibido automaticamente
3. Use a barra lateral para fazer upload de um novo arquivo Excel

## Formato do Excel
- Coluna CLIENTE: Nome do cliente
- Coluna INDICAÇÕES: Número de indicações
